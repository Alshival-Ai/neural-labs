# Neural Labs App

The primary project documentation now lives at the repository root:

- see [`../README.md`](../README.md) for setup, onboarding, startup with `./start.sh`, and the full technical reference

This directory contains the actual Next.js application.

## Local App Commands

If you are working directly on the app:

```bash
npm install
npm run dev
```

Other useful commands:

```bash
npm run typecheck
npm run build
npm run start
```

The app expects the repository-root `.env` file to exist.
